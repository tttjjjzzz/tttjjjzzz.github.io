CAN Bus to Fibre Optic Conversion Module

Overview

Device Function

Enables remote I/O communication by converting CAN bus signals into optical signals (CAN and Fibre Optic Transceivers).

Application

This device allows communication between a controller and remote nodes when standard wiring is impractical (Industrial Settings).

Use Case

Used for an Industiral Control System

My Contributions

For this project, I collaborated on the module's schematic and PCB design using Cadence OrCAD and Allegro, ensuring alignment with overall product specifications. As part of the design process, I researched and selected suitable components, including for both CAN communication and fiber-optic signal conversion ICs/Modules, to meet performance and reliability needs.

Following the initial development, we carried out hardware verification testing and used the results to refine and optimize the PCB layout to meet vertification requirements.
